# CRUD Mahasiswa UTM (PHP Prosedural + MySQLi)

Aplikasi CRUD data mahasiswa Universitas Trunojoyo Madura menggunakan PHP Prosedural, MySQLi, dan Tailwind CSS.

---

### Fitur

- Tambah, Ubah, Hapus, dan Cari Mahasiswa (nama/NIM/jurusan)
- Pagination (5 data per halaman)
- Upload foto (JPG, JPEG, PNG; maks 2MB)

### Persyaratan

- XAMPP (PHP 7.4+), MySQL/MariaDB
- Folder `uploads/` dapat ditulis (writable)

### Setup

1. Import `db.sql` ke MySQL.
2. Letakkan folder proyek di `htdocs` (contoh struktur saat ini: `htdocs/tugaspaw-main/tugaspaw-main`).
3. Ubah kredensial database di `db.php` jika perlu (default: db `db_crud_students`).
4. Pastikan folder `uploads/` ada dan dapat ditulis.

### Menjalankan

- Akses: `http://localhost/tugaspaw-main/tugaspaw-main/index.php`
  - Jika Anda memindahkan isi proyek ke `htdocs/tugaspaw-main` langsung, akses: `http://localhost/tugaspaw-main/index.php`

### Catatan

- Jenis file upload: JPG, JPEG, PNG
- Ukuran maksimal: 2MB
- Pagination: 5 data/halaman
